# Character References
Uppercase means press button, lowercase means release button
Example: The 'A' character tells the Ardunio to press the A button until it detects the 'a' character, then it will release the a button.

Note: ButtonNames.py has all the characters as varibles.
## Joycon Controls
#### Buttons
A,a : A button  
B,b : B button  
Y,y : Y button  
X,x : X button  
T,t : L button  
Y,y : R button  
F,f : zL button  
G,g : zr button  
Q,q : Left Stick Button  
E,e : Right Stick Button   
H,h : Home
V,v Capture   
+,= : plus button   
-,_ : minus button   



#### Joystick
U,u : Left stick Up  
D,d : Left stick Down   
L,l : Left stick Left   
R,r : Left stick Right 

I,i : Right stick Up
K,k : Right Stick down
J,j : Right Stick Right
M,m : Right Stick Left
